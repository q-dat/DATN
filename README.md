## Run Project Here

Clone the project

```bash
  git clone https://github.com/q-dat/Train-Ticket-Booking.git
```

Go to the project directory

```bash
  cd Train-Ticket-Booking
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn dev
```
