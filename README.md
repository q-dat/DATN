## Run Project Here

Clone the project

```bash
  git clone https://github.com/q-dat/FE-Train-Ticket-Booking-Managerment.git
```

Go to the project directory

```bash
  cd FE-Train-Ticket-Booking-Managerment
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn dev
```
